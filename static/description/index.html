tab navigation for odoo

Usage

To use this feature you just have to drag and drop the tab navigation from the structure building block into your content (e.g. below)

Customization

after dropping the snippet you can customize the snippet with the right click customize menu of the website builder. You can can the style of the navigation, add some more tabs or remove not needed tabs. The pictures below illustrating the options you will get after dropping it.

Fill it with Content

You can just drag and drop some content or structure block from the website build into the just created tab navigation. Just drop your new block in the content section of the tab navigation and fell free to edit it like you want. You also able to edit the placeholder content you will see when you drop the snippet.
